Creating lists in Markdown:

Ordered Lists can be created like so:
```
1. List Item 1
2. List Item 2
    - Sub Item 1
    - Sub Item 2
3. List Item 3
```
And for unorderd lists:
```
- Item 1
- Item 2
    - Nested Item 1
    - Nested Item 2
- Item 3
```
